## 项目简介

AI 起名大师是一款智能起名工具，为宝宝、公司、产品、游戏ID、宠物等提供创意名字。

## 核心功能

- 👶 **宝宝起名**：含寓意、字义、五行分析
- 🏢 **公司命名**：品牌名称建议
- 📱 **产品命名**：App、网站名称
- 🎮 **游戏ID**：炫酷个性昵称
- 🐾 **宠物名字**：可爱的宠物名

## 技术栈

- HTML/CSS/JavaScript
- Tailwind CSS
- OpenAI API
- 阿里云 ESA EdgeKV

## 本地运行

直接用浏览器打开 `index.html` 即可使用。
